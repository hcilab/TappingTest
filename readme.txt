Tapping Test

How many times can you click the mouse in a 10-second interval? While seemingly a simple task, performance on this test can be used as a measure of reaction time as well as cognitive functioning and cognitive performance more generally, providing insight about autonomic state.

To run the test, simple lauch the main file in processing. The timer will begin on your first click, and remaining time is displayed in a progress bar along the top of the program window.

Be sure to keep the mouse cursor within the program window throughout the test, as mouse click outside the window will not be detected.

Output is written to the file `clicks-<timestamp>.csv`, with one row per each click performed.
